Install Prerequisites

  1. Java 1.5 or later.
  2. Ant 1.6 or later. (Older versions may also work)
  3. Eclipse 3.2 or later. These instructions assume version 3.2.2, but other versions should behave similarly.

Install Orc

  1. Launch Eclipse
  2. Go to File -> New -> Project ...
  3. Open CVS, choose Projects from CVS, and click Next >
  4. Choose Create a new repository location and click Next >
  5. Fill out the Repository Location Information as follows:
    Host
    orc.cvs.sourceforge.net
    Repository path
    /cvsroot/orc
    User
    anonymous
    Password
    (leave blank)
    Connection type
    pserver
  6. Choose Use a specified module name, enter OrcJava, and click Finish

Running Orc Programs

  1. Within Eclipse, open the Orc program you wish to run.
  2. Go to Run -> Run ...
  3. Right-click Java Application and select New
  4. Fill out the Main form as follows:
    Name
    Orc
    Project
    OrcJava
    Main class
    orc.Orc
  5. Click the Arguments tab and fill out the form as follows:
    Program arguments
    ${resource_loc}
  6. Click Apply
  7. Click Run

You only have to perform the above steps once. Then whenever you want to run an Orc program:

  1. Select the program (you can open it or simply select it in the Package Explorer or Navigator).
  2. Go to Run -> Run History -> Orc (or use the dropdown menu of the run icon in the toolbar).
