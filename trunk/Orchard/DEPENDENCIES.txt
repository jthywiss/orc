This file lists all of the bundled third-party dependencies.  Each dependency
includes the following information:

  FILES: the name of the JAR file(s)
PROJECT: the project from which the JAR file was derived
VERSION: the version of the bundled code
    URL: the URL for the project
LICENSE: the license on the bundled code
   TYPE: build = used for compilation only, not at runtime.
         link = linked against but not extended.
         extend = classes are extended.
         modify = we bundle modified versions of the code.
  NOTES: any notes about why the dependency is needed
