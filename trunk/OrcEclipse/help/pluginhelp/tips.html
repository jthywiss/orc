Tips and Tricks

The following tips and tricks give some helpful ideas for increasing your productivity with Orc. See also Platform Tips and Tricks for general Eclipse tips. They are divided into the following sections:

  • Editing
  • Running

Editing

Which parenthesis matches this one? Move your cursor to the right of a closing parenthesis (or bracket or brace). A box will be drawn around the matching open parenthesis.
I see red reverse highlighted parts of my source code. These are characters that are not valid Orc source code characters. One common source is cut-and-pasted text from e-mail or Web pages. Delete the red reverse highlighted characters.

Running

Nothing runs when I press the Run button or menu item. Eclipse is easily confused about what the "currently selected" file is. In the Project Explorer view, select the Orc program file you want to run, and try again.
My program is not terminating. Press the square red Terminate button on the Console view.
