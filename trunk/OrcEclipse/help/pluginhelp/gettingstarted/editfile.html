Editing the File

  1. If needed, in the Project Explorer, expand the project and folders containing the Orc file of interest.
  2. Double-click the Orc file you wish to view and edit.
  3. Notice the syntax highlighting. Different kinds of elements in the Orc source are rendered in unique colors.
  4. Look at the Outline view. It displays an outline of the Orc file including the include declarations, definitions, site declarations, class declarations, type declarations and val declarations. The Outline view uses icons to annotate Orc elements.
