Run an Orc Program

In this section, you will run an Orc program. Orc programs are run by the Orc runtime engine, which provides the concurrency feature of Orc, and manages communication with sites.

Running the Program

  1. In the Project Explorer, select the Orc program you wish to run. If you need an example, cut and paste the following into a new Orc file: (1 | 2 | 3) >x> x+40
  2. Pick the menu item Run > Run.
  3. Observe the output in the Console view. Runtime error messages will also be displayed in the console.
  4. Pick the menu item Run > Run Configurations..., and observe that a default Orc Program run configuration has been created. Run configurations store settings that control the execution of a program. Other Orc run configurations can be created and used via the Run > Run As... menu item. The next section covers changing these settings.
  5. Run a long running program, such as Rtimer(15000) (a 15 second delay). Press the square red Terminate button on the Console view. This button terminates the program when pressed.
