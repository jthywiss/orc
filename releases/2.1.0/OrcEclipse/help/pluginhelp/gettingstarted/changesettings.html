Change Orc Configuration Settings

In this section, you will observe the two locations in the Eclipse user interface where settings are stored that affect Orc compilation and execution. Settings that affect compilation are associated with an Orc project and are stored as project properties. Settings that affect execution are stored separately as run configuration settings, which are normally stored in your Eclipse workspace.

Changing Project Properties

  1. In the Project Explorer view, select the project which you wish to change.
  2. Pick the menu item File > Properties.
  3. In the Properties window, select the Orc Project Settings category.
  4. Change the settings shown.
  5. Press OK.

Changing Run Configuration Settings

  1. In Eclipse, pick the menu item Run > Run Configurations....
  2. If necessary, expand the Orc Program category.
  3. Select a run configuration under the Orc Program category.
  4. Change the settings shown.
  5. Press Close to save the settings for use later, or press Run to save the settings and use them to run the currently selected Orc program.
