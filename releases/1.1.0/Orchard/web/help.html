User Interface Controls

"Run" buttonRun the program
"Stop" buttonStop the program
"Close" buttonClose the output panel
Drag the grey barResize the program area (make it taller or shorter)

Keyboard Shortcuts

The following shortcuts are available while the program editing area has focus.
Ctrl-EnterRun the program
EscStop the program
Ctrl-/Comment/uncomment the current line or selection
TabAlign indentation with previous line
Ctrl-ZUndo last edit
Ctrl-YRedo last edit
Ctrl-LRe-highlight the program (use if the syntax highlighting becomes incorrect)
F1Display this help

Known Issues

  • Firefox 2 is known to have problems when pasting program text. If you encounter odd behavior when cutting and pasting, use Ctrl-L to re-highlight after pasting.
