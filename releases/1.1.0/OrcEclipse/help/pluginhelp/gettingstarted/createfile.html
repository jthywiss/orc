Create an Orc File

In this section, you will create a new Orc file. There are two kind of Orc source code files: Orc programs (with names ending in .orc), and Orc include files (with names ending in .inc). Orc programs are runnable, but include files are not because they do not have a goal expression.

Creating the File

  1. In Eclipse, select the project or folder you wish to contain the new file.
  2. Select the menu item File > New Orc File to open the New File wizard.
  3. Type a name for the file, ending in .orc, and press Finish.
