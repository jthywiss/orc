Create an Orc Project

In this section, you will create a new Orc project. Unlike other language projects in Eclipse, an Orc project does not build to a single executable. Each Orc program is separately run, so an Orc project is more like a collection of similar or related programs.

Creating the Project

  1. In Eclipse, select the menu item File > New > Project... to open the New Project wizard.
  2. Expand the Orc category, select Orc Project, and press Next.
  3. Type a name for the project, and press Finish.
  4. If you are not in the Orc perspective, Eclipse will ask if you want to switch to it. Answer yes.
