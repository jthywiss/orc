Getting Started Developing Orc programs with the Orc Eclipse Plug-in

Here are brief tutorials on basic use of the Orc Eclipse plug-in. They assume you are familiar with the Eclipse integrated development environment. For help with the Eclipse IDE, see the Workbench User Guide.

The tasks with tutorials available are:

  • Create an Orc project
  • Create an Orc file
  • Edit an Orc file (describes problems and the outline view)
  • Run an Orc program
  • Change Orc configuration settings

To learn about the Orc programming language, see the Orc Language User Guide, and Orc in 15 Minutes and Try Orc on the Orc Web site.
